# Project Context

## Project Goal

The goal of this project is to build a customer and product sentiment analysis dashboard using NLP techniques and AWS S3. The project will clean product review data, analyze sentiment, create processed outputs, upload processed data to AWS S3, and build a Power BI or Streamlit dashboard for business insights.

## Dataset

This project uses the 171k Product Review with Sentiment Dataset.

## Main File

The main dataset file is:

```text
data/raw/sentiment.csv
```

The dataset is already inside the `data/raw` folder.

## Planned Stages

1. Data inspection
2. Data cleaning
3. Sentiment analysis
4. Processed dataset generation
5. AWS S3 upload
6. Dashboard creation
7. Insights and recommendations
